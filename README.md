# CV Manager

CV Manager is a Go-based GraphQL API for managing CV (curriculum vitae) information. It uses MongoDB for data storage and gqlgen for GraphQL schema generation.

## Features

* Create, update, delete, and retrieve CV basic information (BasicInfo).
* Fully implemented GraphQL CRUD API.
* MongoDB integration.
* Dockerized setup for development.

## Project Structure

* `config/`      - Environment configuration and helpers
* `db/`          - Database connection utilities
* `generated/`   - Code generated by gqlgen
* `model/`       - Data models for GraphQL and MongoDB
* `resolvers/`   - GraphQL resolvers
* `schemas/`     - GraphQL schema definitions
* `server.go`    - Application entry point
* `docker-compose.yml` - Docker setup for MongoDB and the app

## Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/ettory-automation/cv-manager.git
   cd cv-manager
   ```

2. Start Docker services:

   ```bash
   docker-compose up -d
   ```

3. Run the application:

   ```bash
   make
   ```

## Usage

Once running, access the GraphQL playground at:

[http://localhost:8080/query](http://localhost:8080/query)

Example GraphQL operations:

* Query all CVs:

```graphql
query {
  BasicInfos {
    id
    firstName
    lastName
    additionalName
    pronouns
    headline
  }
}
```

* Create a new CV:

```graphql
mutation {
  createBasicInfo(basicInfo: {
    firstName: "John"
    lastName: "Doe"
    additionalName: "Smith"
    pronouns: "He/Him"
    headline: "Software Engineer"
  }) {
    id
    firstName
    lastName
  }
}
```

* Update a CV:

```graphql
mutation($id: ID!, $basicInfo: BasicInfoInput!) {
  updateBasicInfo(id: $id, basicInfo: $basicInfo) {
    id
    firstName
    lastName
    headline
  }
}
```
